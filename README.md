# Śrutilekhak (श्रुतिलेखक)

> Local, privacy-first audio transcription browser extension

## What It Does
Transcribes audio from any browser tab in real-time. All processing happens locally on your device - zero cloud dependency.

## Features (Planned)
- ✅ Real-time audio transcription
- ✅ 30+ languages (English + Indian languages)
- ✅ 100% offline/local processing
- ✅ Privacy-preserving (no data leaves device)

## Development Setup

```bash
# Clone repo
git clone https://github.com/ayushsaun24024/srutilekhak.git
cd srutilekhak

# Install dependencies
npm install

# Build for development
npm run build:dev

# Load unpacked extension in Chrome
# chrome://extensions → Load unpacked → Select dist/ folder
